**치지직과 내가 이어지는 맛있는 알림!** 팔로우한 스트리머의 라이브부터 다시보기, 커뮤니티 그리고 치지직 소식까지 한 번에 알려줍니다.

놓치고 싶지 않은 스트리머의 모든 순간을 가장 먼저 확인하세요.  
콘치즈(Con:Chzzk)는 여러분이 팔로우하는 스트리머의 새로운 활동을 감지하여 실시간으로 알려주는 가장 빠르고 편리한 방법입니다.

---

## ✨ 주요 기능

- 🔴 **실시간 라이브 알림**: 팔로우한 스트리머가 방송을 시작하면 즉시 알려드려요.
- 🔄 **방송 정보 변경 알림**: 방송 중 카테고리나 제목이 변경되어도 놓치지 마세요.
- 🎬 **다시보기 & 동영상 알림**: 새로운 다시보기나 동영상이 업로드되면 바로 확인할 수 있어요.
- 🪵 **통나무 파워 기능**
  - **자동 획득 및 알림**: 통나무 파워를 자동 획득하고 알림으로 알려드려요.
  - **파워 표시**: 라이브 페이지의 채팅 입력창 아래에 해당 채널의 현재 통나무 파워를 표시해드려요.
  - **요약 알림**: 일간, 주간, 월간, 연말 동안 획득한 채널별 통나무 파워를 요약하여 알려드려요.
  - **승부예측 알림**: 통나무 파워 승부예측 시작/종료 시 알려드려요.
- 🎉 **파티 알림**: 파티를 생성하거나 참여하면 참여 중인 파티원과 라이브 중인 파티원을 알려드려요.
- 💰 **파티 후원 알림**: 파티가 후원을 시작하면 알려드려요.
- 💬 **커뮤니티 새 글 알림**: 스트리머가 작성한 커뮤니티의 새로운 소식을 놓치지 마세요.
- 🧀 **라운지 소식**: 치지직 라운지의 새로운 소식도 놓치지 마세요.
- 📢 **치지직 배너**: 치지직 메인 페이지의 배너 소식도 놓치지 마세요.
- 💡 **팝업 알림 센터**: 놓친 알림이 있나요? 팝업을 열어 최근 알림 내역을 한눈에 확인하고, 클릭 한 번으로 해당 페이지로 바로 이동할 수 있습니다.
- 🔖 **북마크**: 자주 찾는 스트리머를 북마크하여 채널, 라이브, 동영상, 커뮤니티, 정보로 바로 이동할 수 있어요.
- 🔧 **브라우저 알림 설정**: 라이브, 커뮤니티, 동영상 등 원하는 종류의 알림만 선택해서 받을 수 있도록 브라우저 알림 설정을 제공합니다.
- 🔉 **브라우저 알림 사운드 설정**: 브라우저 알림 사운드를 커스텀할 수 있어요.

### ⌨️ 단축키 지원 (팝업 페이지)
- `W` 또는 `↑` / `S` 또는 `↓`: 알림 목록 위/아래 이동
- `Shift` + `W` 또는 `↑` / `Shift` + `S` 또는 `↓`: 목록의 맨 위/맨 아래로 이동

## 📌 참고 사항

- 치지직에서 팔로우한 스트리머의 알림 설정을 따릅니다.
- 각각의 알림을 OFF할 경우, 해당 "브라우저 알림"만 OFF되며 '최신 알림' 목록에는 알림 메시지가 그대로 기록됩니다.
- 다른 통나무 파워 자동 획득 확장프로그램을 사용 중이라면 통나무 파워 알림이 발송되지 않을 수 있습니다. 
- 통나무 파워 획득은 최초 라이브 방송을 켠 **'하나'의 탭 페이지**가 우선 획득 대상이 되며, 라이브 방송 페이지를 시청 중이거나 음소거가 아닐 경우(탭 음소거는 가능) 지속적으로 통나무 파워 획득이 가능한 것으로 확인됩니다.
- 통나무 파워 요약 알림은 `00시 05분`에 자동으로 발송되며, 해당 시간에 미접속 중이라도 9, 12, 15, 18, 21시에 주기적으로 추가 발송을 시도합니다.

## 🔄 업데이트 방법

자동으로 업데이트가 되지 않는다면, 사용 중인 브라우저의 확장 프로그램 관리 페이지에 접속하여 상단의 **'업데이트'** 버튼을 클릭하세요. 최신 버전의 콘치즈를 바로 사용해 볼 수 있습니다.
- **Chrome**: `chrome://extensions/`
- **Edge**: `edge://extensions/`
- **Whale**: `whale://extensions/`
- **Firefox**: `about:addons` 페이지 상단 톱니바퀴 > '업데이트 확인'
